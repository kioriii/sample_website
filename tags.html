Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings Include a heading (levels 1–6)
p paragraph Include a paragraph of text
table table Include a table
tr table row Include a row of data
th table heading Include a table header
td table data Include a table data cell
div Division Block element to separate a page out
header Header Block element to help in styling
ol Ordered List Creates an ordered list. Use <li> and </li> to add list items.
ul Unordered List Creates an unordered list. Use <li> and </li> to add list items.

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong/bold text <strong>technical sophistication</strong> technical sophistication
a attribute set the attributes including hyperlinks <a href>technical sophistication</a> technical sophistication
img image insert an image <img src="", alt="">technical sophistication> technical sophistication
code code put text in a code formatting <code>technical sophistication<code> technical sophistication
span Span inline element to help in styling <span>technical sophistication<span> technical sophistication

Styling Elements

Tag Name Purpose Example
style=" Style of the tag Allows you to add styling to a tag. Many different arguments can be passed, common ones are font-size, font-style, font-weight etc. <blockquote style="font-size: 24px;"> </blockquote>
p paragraph Include a paragraph of text
